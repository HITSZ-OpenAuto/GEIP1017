## 关于考试
<!-- TOML-SECTION: title="关于考试" -->

<!-- TOML-ITEM: id="item-关于考试-1" -->

本课程考核性质发生过变更：
22、23级为考查课（期末通常为论文）；
24级及以后改为考试课（期末为闭卷考试）。

- 题型有选择、判断、简答、材料分析和论述。
- 选择、判断、简答主要是直接考查对某个知识点的记忆；材料分析需要结合材料理解；24 级的论述考查的是和导论有关的内容，基本就是默写。（导论部分很重要，一定要看！）
